## Your Task

Identify 1-2 concepts from this lesson that would genuinely benefit from animation. NOT every concept needs a visual - only choose concepts where:

1. **Spatial/geometric relationships** are involved (transformations, alignments, flows)
2. **Processes or sequences** that are hard to convey in text
3. **Contrasts** where showing side-by-side helps clarify differences
4. **Abstract concepts** that become clearer with concrete visualization

For each visual opportunity, provide:
- A clear concept name
- Description of what the animation should show (2-3 sentences max, be specific but concise)
- Where it should appear: "core_mechanism", "misconception", "example", or "contrast"
- Why this visual helps learning (1-2 sentences)
- Suggested duration (10-20 seconds)

## Output Format

Return JSON with this exact structure:
{
